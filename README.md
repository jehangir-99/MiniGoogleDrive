# Mini Google Drive

A desktop-based file sharing system developed using C#, WPF, TCP/IP sockets, and SQL Server LocalDB.

## Features
- User Login & Registration
- File Upload & Download
- File Deletion
- Multi-client Support
- Modern WPF UI
- TCP/IP Socket Communication
- Progress Bar
- User-specific File Storage

## Technologies Used
- C#
- WPF
- .NET Framework
- TCP/IP Sockets
- SQL Server LocalDB
